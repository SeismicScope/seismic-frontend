# Level 3: Mode Tokens

Mode tokens are usage-specific colors for different contexts and states. These are the tokens you should use most often in your components.

## Background

Background colors for different container levels.

<ColorPalette>
  <ColorItem
    title="Background"
    colors={{
      "background-default": "var(--background-default)",
      "background-soft": "var(--background-soft)",
    }}
  />
</ColorPalette>

### Usage

```tsx
// Page background
<div className="bg-background">...</div>

// Section background
<div className="bg-bg-soft">...</div>
```

## Surface

Surface colors for cards, panels, and elevated elements.

<ColorPalette>
  <ColorItem
    title="Surface"
    colors={{
      "surface-default": "var(--surface-default)",
      "surface-soft": "var(--surface-soft)",
      "surface-medium": "var(--surface-medium)",
      "surface-disable": "var(--surface-disable)",
      "surface-invert": "var(--surface-invert)",
    }}
  />
</ColorPalette>

### Usage

```tsx
// Card
<div className="bg-card">...</div>

// Muted section
<div className="bg-muted">...</div>

// Secondary element
<div className="bg-secondary">...</div>

// Dark mode / inverted surface
<div className="bg-surface-invert text-content-invert">...</div>
```

## Content

Text and icon colors with different emphasis levels.

<ColorPalette>
  <ColorItem
    title="Content"
    colors={{
      "content-high": "var(--content-high)",
      "content-medium": "var(--content-medium)",
      "content-low": "var(--content-low)",
      "content-disable": "var(--content-disable)",
      "content-invert": "var(--content-invert)",
    }}
  />
</ColorPalette>

### Usage

```tsx
// Heading
<h1 className="text-foreground">Important Title</h1>

// Body text
<p className="text-muted-foreground">Regular paragraph text</p>

// Caption or helper text
<span className="text-content-low">Optional information</span>

// Disabled text
<span className="text-content-low">Disabled option</span>

// Text on dark background
<span className="text-content-invert">White text</span>
```

## Outline

Border and divider colors.

<ColorPalette>
  <ColorItem

```tsx

```

---

## Visual Hierarchy

Mode tokens follow this layering system:

1. **Background** - Base layer (page)
2. **Surface** - Elevated elements (cards, panels)
3. **Content** - Text and icons
4. **Outline** - Borders and dividers

When building components, always use Mode tokens rather than Core or Theme tokens directly for maximum flexibility.
